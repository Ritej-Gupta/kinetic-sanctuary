# Kinetic Sanctuary v3.0: Performance Intelligence Platform

Kinetic Sanctuary is a high-performance, editorial-inspired intelligence dashboard designed for high-achievers. It transforms the utility of habit tracking into a prestigious performance hub, focusing on precision, consistency, and professional-grade monitoring.

## 🏛 Design System: Kinetic Precision (v3.0)

The application adheres to the **Kinetic Precision** aesthetic—a balance between a high-end editorial and a high-performance flight deck.

- **Typography**: A curated pairing of **Manrope (800 weight)** for headlines and **Inter** for technical data.
- **Palette**: Deep surface-container foundations with **Legendary Amber (#f0c060)** accents for peak achievement.
- **Architecture**: A boundary-less "No-Line" system that uses structural surface levels and ambient glows instead of hard borders.

## 🚀 Key v3.0 Features

### 1. AI Coach (Gemini 2.0 Flash)
A persistent intelligence layer that analyzes your protocol velocity and streak data to provide calm, data-driven coaching insights. No generic motivation—just precision guidance.

### 2. Resonance Map (Advanced Analytics)
A sophisticated correlation chart that identifies intersection zones between Physical and Mental focus scores, calculating your real-time "flow resonance" coefficient.

### 3. Protocol Bar (Numeric Performance)
High-fidelity progress tracking for numeric daily goals. Features a legendary growth pulse and overachieve states when protocols exceed their target thresholds.

### 4. Quick Entry Morphing FAB
A glassmorphic floating action button that uses layout animations to morph into a rapid entry sheet, streamlining the check-in process from any view.

### 5. Focus & Recovery Mode
Adaptive UI states that dim non-essential telemetry after 7:00 PM, reordering the dashboard to focus on "Today's Wrap" and "Tomorrow's Protocol" preparation.

## ⚙️ Configuration

To run this application locally:

1. **Environment Variables**:
   Define `VITE_GEMINI_API_KEY` in your environment or `.env.local`.
   ```env
   VITE_GEMINI_API_KEY="your_api_key_here"
   ```

2. **Installation**:
   ```bash
   npm install
   ```

3. **Development**:
   ```bash
   npm run dev
   ```
